* * **[Optimizing the UPGMA Algorithm]((https://github.com/Rafael-jar/Phylogenetic-Tree-Upgma))**
    A group project for my advanced algorithms course where we focused on optimizing the UPGMA algorithm for phylogenetic tree construction. We experimented with four different data structures, successfully reducing the time complexity from a terrible $\theta(n^4)$ down to an efficient **$\theta(n^2log~n)$** by using a custom key class with a combination of `std::map` and `std::unordered_map`.

* **[Bloom Filter Analysis](https://github.com/Rafael-jar/Bloom-Filter-Project)**
    A C++ project for my Randomized Algorithms class. I built a Bloom filter from scratch using double hashing for efficiency. The main goal was to experimentally test the false positive rate against the theoretical formula $k=(m/n)ln(2)$, and I found some interesting anomalies where the theory breaks down at extreme (very small or very large) *n* values.
